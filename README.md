# Community Hub

A full-stack MERN chat application featuring persistent messaging, authentication, and a monorepo architecture for end-to-end type safety.

[View Live Demo](https://chat-app-frontend-ten-gold.vercel.app/)

 ## Key Features

- Secure Auth: JWT-based authentication flow using the jose library.

- Persistent Chat: All rooms, users, and message history stored in MongoDB Atlas.

- Type Safety: Shared TypeScript workspace ensures the Frontend and Backend data models are always in sync.

- Responsive UI

## Tech Stack

- Frontend: React (Vite), TypeScript, React Router, Tailwind CSS.

- Backend: Node.js, Express, Mongoose.

- Database: MongoDB Atlas.

- Deployment: Vercel (Frontend) & Render (Backend)
